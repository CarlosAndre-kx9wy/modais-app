Implementa cards de modais com pesquisa dinâmica e Dark Mode

Este commit adiciona a estrutura inicial do projeto "Modais de Transporte", desenvolvido durante a Imersão Dev com Google Gemini 10ª Edição. A página apresenta informações detalhadas sobre os modais (Rodoviário, Ferroviário, Aéreo, Hidroviário) em um formato de cards.

O foco principal foi em interatividade e usabilidade:

- **Pesquisa Dinâmica (JavaScript):** Permite filtrar os cards em tempo real pelo nome do modal ou pelas tags de características (ex: "Alto Custo", "Flexibilidade").
- **Modo Escuro (CSS/JavaScript):** Implementação de um toggle no cabeçalho para alternar o esquema de cores, melhorando a experiência do usuário.
- **Estrutura Semântica (HTML):** Uso de tags como `<header>`, `<main>`, `<nav>` e `<article>` para acessibilidade e SEO.

Adiciona os arquivos: `index.html`, `style.css` (para estilo) e `script.js` (para as funcionalidades).
